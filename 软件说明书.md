## 一、软件概述

### 1.1 软件简介

智图忆市K线形态分析系统是一款专业的股票技术分析工具，通过量化方法分析K线形态，帮助用户进行历史数据复盘和形态研究。

### 1.2 主要功能

- **形态匹配**：基于余弦相似度算法，在历史数据中寻找与当前走势相似的形态
- **概率统计**：统计历史相似形态的后续表现，计算上涨概率和预期收益
- **形态检测**：自动识别双底、头肩顶、三角形整理等经典技术形态
- **技术指标**：支持MA均线、MACD、RSI等技术指标叠加显示
- **数据导出**：支持CSV格式导出，方便用户进行二次分析

### 1.3 适用人群

- 金融从业者：复盘研究、形态筛选
- 财经内容创作者：可视化素材、案例分析
- 股票投资者：学习技术分析、验证交易策略
- 量化研究者：数据研究、策略验证

1. 访问Python官网下载Python 3.9+

3. 验证安装：
   ```
   python --version
   ```
   显示Python 3.9.x即安装成功

### 3.2 安装依赖包

1. 打开命令行（Windows按Win+R输入cmd）

2. 进入软件目录：
   ```
   cd 软件所在目录
   ```

3. 安装依赖：
   ```
   pip install -r requirements.txt
   ```

在软件目录下执行：
```
streamlit run app.py
```

浏览器会自动打开 http://localhost:8501 显示软件界面。

## 四、使用说明

### 4.1 界面概览

软件界面分为三个区域：
- **左侧边栏**：分析设置区
- **中央区域**：结果显示区
- **顶部标题**：软件名称和说明

### 4.2 选择标的

1. 在左侧边栏点击"选择标的"下拉框
2. 选择要分析的股票或指数
3. 支持的标的包括：
   - 指数：上证指数、深证成指、创业板指等
   - 蓝筹股：贵州茅台、中国平安、招商银行等
   - 科技股：比亚迪、隆基绿能、中芯国际等

### 4.3 设置参数

| 参数 | 说明 | 默认值 | 范围 |
|------|------|-------|------|
| 模板窗口天数 | 用于匹配的历史窗口大小 | 20天 | 5-60天 |
| 后续观察天数 | 统计后续表现的天数 | 20天 | 5-60天 |
| 匹配数量 | 返回最相似的N个形态 | 10个 | 3-20个 |
| 历史数据量 | 用于搜索的历史数据范围 | 800条 | 300-1000条 |

### 4.4 图表设置

可选显示内容：
- ✅ 显示均线 (MA5/10/20)
- ✅ 显示成交量
- ✅ 显示MACD
- ✅ 自动检测常见形态

### 4.5 开始分析

1. 设置好参数后，点击"🔍 开始分析"按钮
2. 等待数据加载和分析（约5-15秒）
3. 结果显示在中央区域

### 4.6 查看结果

分析结果包含：
- **形态匹配图表**：展示当前形态与历史相似形态的对比
- **概率统计卡片**：显示上涨概率、平均涨跌幅等
- **匹配详情列表**：列出所有匹配的历史形态
- **形态检测结果**：自动识别的技术形态

### 4.7 导出数据

1. 在结果页面找到"导出CSV"按钮
2. 点击下载CSV文件
3. 可用Excel或Python进行进一步分析

## 五、功能详解

### 5.1 形态匹配算法

采用余弦相似度算法：
1. 将K线走势归一化为向量
2. 计算与历史每个窗口的余弦相似度
3. 返回相似度最高的N个匹配

### 5.2 概率统计方法

基于历史匹配结果统计：
- 上涨次数 / 总匹配数 = 上涨概率
- 计算各持有期的平均收益
- 计算最大涨跌幅

### 5.3 形态检测规则

| 形态名称 | 检测逻辑 |
|---------|---------|
| 双底 | 两个低点价格接近，中间有反弹 |
| 双顶 | 两个高点价格接近，中间有回调 |
| 头肩顶 | 三个高点，中间最高 |
| 头肩底 | 三个低点，中间最低 |
| 三角形整理 | 高点降低、低点抬高 |
| 均线多头排列 | MA5 > MA10 > MA20 |
| 均线空头排列 | MA5 < MA10 < MA20 |
