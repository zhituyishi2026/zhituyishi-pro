### 二、新增策略回测模块

#### 新增函数：

**`backtest_pattern_strategy(df, pattern_name, holding_days=20)`**
- 参数：
  - `df`: 包含技术指标的数据框
  - `pattern_name`: 形态名称
  - `holding_days`: 持有天数（5-60天可选）
- 返回值：包含以下统计指标的字典
  - `total_trades`: 交易次数
  - `profitable_trades`: 盈利交易次数
  - `win_rate`: 胜率（%）
  - `avg_profit`: 平均收益（%）
  - `max_profit`: 最大收益（%）
  - `min_profit`: 最大亏损（%）
  - `sharpe_ratio`: 夏普比率（简化版）
  - `trades`: 详细交易列表
  - `equity_curve`: 权益曲线

**`show_backtest_results(backtest_result)`**
- 显示回测结果的完整UI组件
- 包含：
  - 策略概要卡片（交易次数、胜率、平均收益、最大收益、夏普比率）
  - 权益曲线（Plotly折线图）
  - 收益分布直方图
  - 交易明细表（日期、买入价、卖出价、收益%）
  - CSV导出按钮

#### UI集成：
- 侧边栏新增"📉 策略回测"可折叠区域
  - 形态选择下拉框
  - 持有天数滑块（5-60天）
  - 开始回测按钮
- 主界面新增"第四部分：策略回测"区块
  - 形态选择下拉框
  - 持有天数滑块
  - 回测按钮
  - 实时显示回测结果

### 三、新增PDF报告生成

#### 新增函数：

**`generate_pdf_report(df, selected_stock, patterns, backtest_result=None, matches=None)`**
- 使用 `reportlab` 库生成专业PDF报告
- 报告内容：
  1. **封面**
     - 标题："智图忆市 分析报告"
     - 标的名称
     - 生成时间
  
  2. **当前行情摘要**
     - 最新价、涨跌幅
     - 最高价、最低价
     - RSI、MACD状态
  
  3. **形态检测结果**
     - 列出检测到的所有形态
     - 包含信号、置信度、描述
  
  4. **形态匹配统计**
     - 历史匹配次数
     - 上涨次数
     - 胜率
     - 平均收益
  
  5. **策略回测结果**（如果执行了回测）
     - 交易次数、胜率
     - 平均收益、最大收益、最小收益
     - 夏普比率
  
  6. **免责声明**

#### UI集成：
- 主界面新增"第五部分：生成分析报告"区块
- "📄 生成PDF报告"按钮
- 自动生成PDF并提供下载
- 文件名格式：`智图忆市分析报告_{标的}_{时间戳}.pdf`

## 使用指南

### 基础使用流程
1. 选择标的（股票/指数）
2. 设置形态匹配参数
3. 点击"开始分析"
4. 查看形态检测和匹配结果

### 策略回测流程
1. 在侧边栏或主界面选择要回测的形态
2. 设置持有天数
3. 点击"开始回测"
4. 查看回测统计、权益曲线、交易明细
5. 导出CSV进行进一步分析

### PDF报告生成流程
1. 完成形态分析和回测（可选）
2. 点击"生成PDF报告"
3. 等待报告生成
4. 点击"下载PDF报告"
