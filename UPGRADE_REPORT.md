## 🚀 使用指南

### 快速开始
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行应用
streamlit run app.py

# 3. 打开浏览器
# http://localhost:8501
```

### 基本流程
1. **选择标的** - 从侧边栏选择股票或指数
2. **设置参数** - 调整形态匹配和图表参数
3. **开始分析** - 点击"开始分析"按钮
4. **查看结果** - 浏览形态检测和匹配结果
5. **回测策略** - 选择形态进行策略回测
6. **生成报告** - 一键生成PDF分析报告

### 高级功能
- **策略回测**：在侧边栏或主界面选择形态和持有天数
- **PDF报告**：包含完整的分析结果和统计数据
- **数据导出**：支持CSV格式导出

1. **数据质量**
   - 确保数据源可用
