# 智图忆市 Pro v2.0

**K线形态量化分析平台** | Python + Streamlit + Plotly

---

## 产品定位

智图忆市 Pro 是一款面向专业投资者和量化研究人员的 K线形态分析工具，基于"历史总会押着韵脚"的核心思想，通过 AI 驱动的形态匹配和概率统计，帮助用户发现历史相似行情、评估潜在风险与机会。

---

## 核心功能

### 1. 形态匹配
- 基于余弦相似度的滑动窗口形态匹配
- 可调节模板窗口（5-60天）和观察期（5-60天）
- 在 800+ 天历史数据中精准检索 Top-N 最相似形态
- 匹配结果展示匹配段 + 后续走势全貌

### 2. 形态自动检测
- 双底 / 双顶形态识别
- 上升三角形 / 下降三角形识别
- 均线多头排列 / 空头排列判断
- RSI 超买超卖检测
- MACD 金叉死叉信号

### 3. 概率统计分析
- 历史胜率计算（上涨 vs 下跌）
- 多持有期收益分析（5日/10日/20日）
- 收益分布直方图
- 量价波动率统计
- 详细 CSV 数据导出

### 4. 专业级K线图表
- Plotly 交互式 K 线图（支持缩放、拖拽）
- MA5 / MA10 / MA20 均线叠加
- MACD 指标（附图）
- 成交量柱状图 + 均量线
- 匹配区域高亮标记

### 5. 多数据源
- 新浪财经 API（主数据源）
- AKShare（备用数据源，双保险）
- 涵盖 A股主要指数 + 蓝筹股 + 科技股 + 金融科技股
